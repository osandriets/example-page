# webpack project 

 
## Environment setup 

```sh
  $ npm install
```

## Development

Start the Webpack server (includes live reloading when you change files):

```sh
  $ npm start
```

Open [http://localhost:3001](http://localhost:3001) in a browser. `./src/main.js` is the entry point.


## Bundling 

```sh
  $ npm run bundle
```
